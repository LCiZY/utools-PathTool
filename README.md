# utools-PathTool
utools(https://u.tools/) 平台的一个插件

## 插件实现了以下功能：
1. 复制文件(夹)路径
2. 复制文件(夹)名称（不带后缀）
3. 复制文件(夹)名称（带后缀）
4. 将Windows的文件复制为WSL中可用的文件路径
5. WSL和Windows文件路径互相转换
6. 将路径中的斜杠和反斜杠互相替换
6. 将路径中的单反斜杠转换为双反斜杠(反过来也支持)



# 安装

- 复制upx文件后唤出utools即可看到安装选项
- 或在插件市场搜索**路径转换**(id: zl1r37rp)
